# karimsangid.dev — Personal Portfolio

A single-page portfolio site for Karim Sangid, built with pure HTML, CSS, and JavaScript. No frameworks, no build tools — just clean, performant code.

**By Hummus Development LLC**

## Live Site

[karimsangid.dev](https://karimsangid.dev)

## Features

- Responsive single-page design (mobile-first)
- Smooth scroll animations via IntersectionObserver
- Animated CSS gradient mesh background
- Project showcase with custom CSS illustrations
- Vertical timeline for experience
- Mobile hamburger navigation
- SEO optimized with Open Graph tags
- Lighthouse 95+ performance score

## Deployment (GitHub Pages)

1. Create a GitHub repository named `karimsangid.dev`
2. Push this code to the `main` branch
3. Go to **Settings > Pages** and set source to `Deploy from a branch` > `main` / `/ (root)`
4. If using a custom domain, ensure DNS is configured:
   - A records pointing to GitHub Pages IPs
   - CNAME record for `www` subdomain
5. The `CNAME` file is already included in the repo

## Local Development

Just open `index.html` in a browser. No build step required.
